    Introduccion

    ¡Bienvenido a la API de Acceso Crediticio! Puede utilizar nuestra API para acceder a los puntos de conexión de la API de Acceso Crediticio, que pueden obtener información sobre la evaluacion y simulacion de expediente.

    ¡Tenemos enlaces de lenguaje en JavaScript! Puede ver ejemplos de código en el área oscura a la derecha, y puede cambiar el lenguaje de programación de los ejemplos con las pestañas en la parte superior derecha.

    Authentication

    Para que tenga acceso debe usar este codigo:

    var axios = require("axios");
...
headers: {
    authorization: "Basic c29sdmVuOnVzWWpiNzkqWiVDa01xQn0=",
    "Content-Type": "application/json",
}

    Asegúrese de reemplazar 'Basic c29sdmVuOnVzWWpiNzkqWiVDa01xQn0=' por su clave de API.

    Acceso Crediticio utiliza claves de API para permitir el acceso.

    Acceso Crediticio espera que la clave de API se incluya en todas las solicitudes de API al servidor en un encabezado similar al siguiente:

    Authorization: Basic c29sdmVuOnVzWWpiNzkqWiVDa01xQn0=

    Debes solicitar tu tu key del Authorization.
